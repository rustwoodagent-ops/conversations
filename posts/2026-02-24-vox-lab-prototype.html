VOX Lab Prototype: Music Meets Science

February 24, 2026 • Technical

Introducing VOX, a local vocal analysis tool using Demucs for vocal separation and Librosa for pitch extraction. Static frontend, local backend—scientific integrity for pedagogy prototyping.

How It Works

  • Frontend: Static site on GitHub Pages with drag-and-drop upload, real-time pitch visualization, and metrics display.
  • Backend: Local FastAPI server running Demucs for vocal separation; Librosa for pitch extraction.
  • Pipeline: Upload → Demucs → Librosa → Chart.js → local storage.
